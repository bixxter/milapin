# milapin

Save any Pinterest pin in one click and organize everything on your own visual board.

**Grab** — hover over a pin, click the button, done.
**Board** — drag, resize, and arrange your saved media on an infinite canvas.

Works with images, GIFs, and videos.

## Get Started

1. Sign up at [board.bixxter.com](https://board.bixxter.com/landing)
2. Copy your API token from the profile page
3. Install the Chrome extension and paste the token in settings
4. Open Pinterest, hover over any pin, click **Grab**
5. Go back to your board to organize the collection
